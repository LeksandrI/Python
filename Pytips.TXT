eval() // позволяет исполнять код, который записан в виде строки. 
Использование функции eval() считается небезопасным в том смысле, что она исполняет
любой код, заключенный в строке, не проверяя при этом, является ли код хорошим, правильным.

literal_eval()

Вызов функции from ast import literal_eval // позволяет написать код в виде строки например список iteral_eval("[1,2,3]")
получим [1, 2, 3]


list - получим список обозначается []
tuple - получим кортеж, почти тоже самое, что и список только нельзя изменять его значения обозначается в ()

*имя - звездочка позволяет не ставить дополнительные скобки, как бы записывает значчение в которое в скобках в скобки
